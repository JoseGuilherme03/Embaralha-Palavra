# Baixando e configurando

- Abra um terminal:

    `Control"` + `Alt` + `t`

- Clone o repositório:

```bash
git clone https://github.com/JoseGuilherme03/Embaralha_Palavra-Game.git
```

- Entre na pasta:
  
```bash
cd Embaralha_Palavra-Game
```

- Instale as dependências:

```bash
poetry install
```

- Ative o ambiente virtual:

```bash
poetry shell
```

- Execute o script:

```bash
python embaralha.py
```

- Se preferir, abra o vscode:

```bash
code .
```
